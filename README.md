# Sentiment-Analysis
Sentiment Analysis using lexicon based approach on product reviews (Mobile Phones)

This project reduces the time spent by the customer and developer for reviewing feedbacks. It helps to identify drawbacks of a product as inferred from the customer feedback. The project sorts the reviews on the basis of reviewsâ€™ polarity which helps in understanding customer satisfaction. Analyzing the reviews also reduces the efforts of selling company and the customer for the summary of product reviews.

The Project includes:
  	1. Pre-Processing
      1.a. Removing Stopwords
      1.b.Lemmatizing
      1.c. POS tagging
    2. Feature Selection
    3. Polarity Check
    4. Pros and Cons
    5. Graphical User Interface
    6. Visual Representation
